gfl by pisces

The GFL framework is light component package based on pure flash platform, and it isn't includes classes in the adobe mx package. So, that has higher performance than it is based on mx.core.UIComponent or fl.core.UIComponent. Also, it can use css style declarations on the pure flash platform excludes mx package. GFL framework helps you can modify external css file to change application views at runtime. You can build the light applications that includes free light components, chart components.
